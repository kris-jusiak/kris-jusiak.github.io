    <b>14.5.3 [temp.variadic] p4:</b> <br />

    Pack expansions can occur in the following contexts: <br />
    <ul>
        <li>In a function parameter pack (8.3.5); the pattern is the parameter-declaration without the ellipsis.</li>
        <li>In a template parameter pack that is a pack expansion (14.1):</li>
            <ul>
                <li>if the template parameter pack is a parameter-declaration; the pattern is the parameter-declaration without the ellipsis;</li>
                <li>if the template parameter pack is a type-parameter with a template-parameter-list; the pattern is the corresponding type-parameter without the ellipsis.</li>
            </ul>
        <li>In an initializer-list (8.5); the pattern is an initializer-clause.</li>
        <li>In a base-specifier-list (Clause 10); the pattern is a base-specifier.</li>
        <li>In a mem-initializer-list (12.6.2); the pattern is a mem-initializer.</li>
        <li>In a template-argument-list (14.3); the pattern is a template-argument.</li>
        <li>In a dynamic-exception-specification (15.4); the pattern is a type-id.</li>
        <li>In an attribute-list (7.6.1); the pattern is an attribute.</li>
        <li>In an alignment-specifier (7.6.2); the pattern is the alignment-specifier without the ellipsis.</li>
        <li>In a capture-list (5.1.2); the pattern is a capture.</li>
        <li>In a sizeof... expression (5.3.3); the pattern is an identifier.</li>
    </ul>
